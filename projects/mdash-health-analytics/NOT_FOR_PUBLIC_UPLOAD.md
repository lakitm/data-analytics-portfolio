# Do Not Upload Publicly

Do not upload the following materials to a public GitHub repository:

- raw dengue records;
- row-level health data;
- patient-level or barangay-level sensitive records if access was restricted;
- full internal notebooks with private file paths;
- system credentials, Supabase keys, database URLs, API keys, or admin accounts;
- full unpublished thesis manuscript unless all co-authors, adviser, school, and data providers allow it;
- screenshots showing admin accounts, private tables, internal logs, or restricted modules;
- deployment-ready source code if ownership is shared with the team, institution, or local government.

Recommended public version:

- case study summary;
- sanitized method overview;
- synthetic data schema;
- aggregate result table;
- cropped or redacted screenshots;
- high-level architecture diagram.
