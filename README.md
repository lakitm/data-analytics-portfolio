## Featured Projects

### 1. MDASH: Health Analytics & Dengue Forecasting Dashboard
**Focus:** forecasting, health analytics, time-series modeling, dashboard strategy, decision support  
**Tools:** Python, Pandas, Scikit-learn, XGBoost, TensorFlow/Keras, Statsmodels, Next.js, Supabase, Cube.js, Apache ECharts

MDASH is a web-based health analytics dashboard concept for dengue surveillance in the City of Malolos. The project integrated barangay-level trend visualization, short-term forecasting, health facility mapping, public advisories, and administrative controls. For the forecasting component, multiple models were compared using weekly dengue and climate variables, with Random Forest selected as the best-performing deployment model under rolling evaluation.

[View MDASH case study](projects/mdash-health-analytics/)

### 2. Dirty Housing Data: Cleaning & Linear Regression Workflow
**Focus:** data cleaning, preprocessing, regression modeling, model evaluation  
**Tools:** Python, Pandas, Scikit-learn, Matplotlib

This project demonstrates an end-to-end workflow for handling a messy housing-price dataset: inspection, missing-value handling, duplicate removal, feature preparation, preprocessing pipelines, regression modeling, and evaluation using MAE, MSE, and R².

[View dirty housing regression project](projects/dirty-housing-regression/)
